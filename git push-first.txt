1. ad .....                              파일 위치
2.  git init                             로컬 저장소 생성
3. git status                            파일 상태 확인
4. git commit -m                         간단한 메모 남기기

5. git remote add origin https://...     github 주소 연결
6. git branch                            연결가지 상태 확인
7. git branch -m master main             master에서 main으로 변경
8. git pull origin main --rebase         원격 저장소의 변경 사항을 가져와 병합
9. git push origin main                  푸시
